Reconnaissance

Reconnaissance involves gathering information about the target system, such as domain names, IP addresses, and network infrastructure. It is divided into two parts: Active and Passive Reconnaissance.

Scanning (🔎)

Scanning involves identifying vulnerabilities in the target system using tools like Nessus, Nmap, and others to find weaknesses.

Exploitation (💥)

In this phase, the attacker attempts to exploit vulnerabilities to gain unauthorized access to the system.

Post-Exploitation (🔒)

After gaining access, the attacker escalates privileges and maintains access. This phase may involve lateral movement within the network.

Reporting (📊)

At the end of the test, a report is generated detailing vulnerabilities, exploitation methods, and recommendations for improvement.
