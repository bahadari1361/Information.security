Enterprise Cybersecurity Architecture

Firewall
Protects the network perimeter by filtering traffic.
IDS/IPS
Detects and prevents intrusions within the network.
VLAN
Segregates network traffic for enhanced security.
VPN
Provides secure, encrypted access for remote users.
SPAN Port
Mirrors traffic for monitoring by security tools.
Encryption
Secures data at rest and in transit.
DLP
Prevents data leakage and monitors sensitive information.
