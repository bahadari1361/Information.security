Tools & Techniques Used in the Vulnerability Assessment

Comprehensive Tools and Techniques for Security Assessments at the Central Bank of Afghanistan

1. Wireshark

Tool Name Function Techniques Used
Wireshark Packet Analysis
  • Captured and analyzed network traffic for security issues.
  • Monitored for unencrypted sensitive data or unauthorized access attempts.
  • Investigated incidents by analyzing network packets in real-time.

2. Nmap

Tool Name Function Techniques Used
Nmap Network Scanning
  • Used for open port scanning and service detection.
  • Performed OS fingerprinting to identify running systems.
  • Utilized Nmap Scripting Engine (NSE) for vulnerability detection.

3. Zenmap

Tool Name Function Techniques Used
Zenmap Network Mapping & Visualization
  • Provided a GUI for Nmap for easier visualization of scan results.
  • Mapped out network topology to identify possible vulnerabilities.
  • Helped prioritize vulnerabilities based on scan results.

4. OWASP ZAP

Tool Name Function Techniques Used
OWASP ZAP Web Application Security Testing
  • Automated scans to find vulnerabilities like XSS and SQL Injection.
  • Performed manual testing to validate complex vulnerabilities.
  • Integrated with continuous security monitoring processes.
