Overview

As the Information Security Manager at the Central Bank of Afghanistan, I was responsible for designing, implementing, and managing a comprehensive set of security measures to protect the bank’s sensitive data, networks, and IT infrastructure. This portfolio outlines my work in developing security policies, business continuity plans, incident response strategies, conducting IT audits, penetration testing, and security assessments to ensure the organization’s resilience against evolving cyber threats. Additionally, I played a key role in the deployment of ISO 27001 to align the bank's security framework with international best practices.

Information Security Policy Development

  • Network Security Policy: Designed guidelines for firewalls, VPNs, and network segmentation.
  • Email Security Policy: Established encryption and anti-phishing protocols.
  • User Management and Password Policy: Implemented multi-factor authentication (MFA) and password management guidelines.
  • Data Disposal Policy: Defined secure methods for data destruction.
  • Internet Usage Policy: Instituted controls to block malicious sites.
  • FTP Server Policy: Established secure file transfer protocols.
  • Patch Management Policy: Developed a patch management framework for system updates.
  • Remote Desktop Policy: Defined secure access through VPNs and strong authentication.

Penetration Testing

  • External and Internal Penetration Testing: Simulated attacks to find network vulnerabilities.
  • Web Application Security Testing: Conducted assessments for SQL injection and XSS vulnerabilities using tools like Acunetix and OWASP ZAP.
  • Reporting and Remediation: Collaborated with teams to address and patch identified vulnerabilities.

Conclusion

As the Information Security Manager at the Central Bank of Afghanistan, I led the development and implementation of security strategies to protect the bank’s critical IT assets. From creating robust security policies to hands-on tasks like penetration testing and incident response planning, my work helped safeguard the bank against evolving cyber threats.
