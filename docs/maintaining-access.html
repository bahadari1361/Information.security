Maintaining Access

The Maintaining Access phase focuses on ensuring persistent access to the compromised system or network. In this phase, the ethical hacker attempts to maintain their access over an extended period without being detected. This can involve installing backdoors, rootkits, or using other techniques that provide long-term access.

Objectives

Maintaining access involves several key goals:

  • Ensure continued access to the target system even after reboots or system updates.
  • Install tools or create accounts that allow for repeated access to the system.
  • Expand control across the network by moving laterally to other systems.
  • Gather sensitive data and maintain covert operations to avoid detection.

Tools and Techniques

  • Netcat: A versatile tool for creating backdoors and maintaining reverse shells.
  • Metasploit: Used for exploiting vulnerabilities, maintaining access, and post-exploitation tasks.
  • Mimikatz: A tool for extracting credentials and other sensitive data from the system.
  • PowerShell Empire: A post-exploitation framework for lateral movement and persistence on Windows systems.
  • Cobalt Strike: A tool for post-exploitation activities, including persistence and lateral movement in a target environment.

Common Techniques

Some common techniques used during the Maintaining Access phase include:

  • Backdoors: Installing a backdoor allows the tester to bypass security controls and regain access to the system at a later time.
  • Rootkits: A rootkit is used to hide malicious activities and maintain control over a system without being detected.
  • Credential Dumping: Extracting credentials from the system to enable further access or lateral movement.
  • Lateral Movement: Moving from one compromised system to another within the network to extend control and gather more data.
  • Persistence Mechanisms: Setting up mechanisms that will ensure continued access even if the system is rebooted or patched.

Findings Table

Finding Risk Level Details Mitigation
Backdoor Installed High Access maintained through a hidden backdoor, allowing repeated exploitation. Remove unauthorized access, change credentials, and enhance monitoring.
Credential Dumping High Tested tools successfully dumped sensitive credentials. Change passwords, enforce multi-factor authentication, and use credential vaults.
Lateral Movement Medium Tested tools allowed moving to other systems in the network. Segment networks, isolate critical systems, and use proper access controls.
