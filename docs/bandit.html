Bandit Level 0 to Level 1 Challenge

A Step-by-Step Guide to Accessing the Next Level Using SSH

Overview

This challenge is part of the **OverTheWire Bandit** game, a series of levels designed to enhance penetration testing and Linux command-line skills. The goal of **Level 0 to Level 1** was to find the password for the next level, stored in a file called `readme` in the home directory, and use it to log into the next level using **SSH** (Secure Shell).

Skills Utilized

  • SSH (Secure Shell)
  • Linux Command Line
  • File Navigation
  • Cybersecurity Fundamentals
  • Penetration Testing

Step-by-Step Breakdown

Step 1: Initial SSH Login to Bandit Level 0

To begin, I connected to the Bandit server via SSH. The command used is as follows:

Bahadari@Mac-Terminal:
ssh bandit0@bandit.labs.overthewire.org -p 2220

The login was completed with the password provided in the challenge instructions.

Step 2: Navigating to the Home Directory

After logging in, I verified that I was in the home directory where the `readme` file was located. To confirm, I used the `pwd` command:

Bahadari@Mac-Terminal:
pwd

The output confirmed the home directory:

/home/bandit0

If necessary, I used the following command to explicitly navigate to the home directory:

Bahadari@Mac-Terminal:
cd ~

Step 3: Listing Files in the Home Directory

Next, I listed all files in the home directory to find the `readme` file:

Bahadari@Mac-Terminal:
ls -l

The file `readme` was found in the listing, and I proceeded to view its contents.

Step 4: Accessing the `readme` File

To retrieve the password for **Bandit Level 1**, I used the `cat` command:

Bahadari@Mac-Terminal:
cat readme

The content displayed the password required for the next level.

Step 5: SSH Login to Bandit Level 1

With the password for **Bandit Level 1**, I logged in using the following command:

Bahadari@Mac-Terminal:
ssh bandit1@bandit.labs.overthewire.org -p 2220

I entered the password when prompted, successfully logging into **Bandit Level 1**.

Outcome and Learning

Successfully completing the **Bandit Level 0 to Level 1** challenge demonstrated my proficiency with basic Linux command-line utilities (`ls`, `cat`, `pwd`, etc.) and my ability to establish SSH connections and retrieve sensitive information from files. This challenge reinforced fundamental cybersecurity skills, including file navigation, SSH access, and password retrieval in a penetration testing context.
