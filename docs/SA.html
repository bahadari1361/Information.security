Security Assessment for the Central Bank of Afghanistan

As the Information Security Manager at the Central Bank of Afghanistan, I led comprehensive security assessments to evaluate the integrity and security posture of the bank's IT infrastructure. These assessments were essential for identifying potential vulnerabilities and ensuring the protection of sensitive financial data. Using a combination of manual and automated testing tools, I conducted thorough penetration testing, vulnerability scans, and risk analysis to assess the security landscape and provide actionable recommendations for improvement.

  • Identify vulnerabilities in the bank's internal and external networks.
  • Evaluate the security of critical systems and applications.
  • Ensure compliance with industry standards.
  • Develop and implement strategies to address identified risks.

Nessus: Vulnerability scanning and risk assessment tool to identify known security flaws and weaknesses across systems and networks.

Nmap: Network discovery and security auditing tool that helps identify open ports, running services, and network configurations.

Kali Linux: A penetration testing platform equipped with numerous tools for vulnerability assessment and exploitation.

Acunetix: Automated web application security scanner that detects vulnerabilities like SQL injection, XSS, and other web-based security risks.

Wireshark: Network protocol analyzer for capturing and analyzing network traffic to identify potential vulnerabilities in network protocols and communication.

OWASP ZAP (Zed Attack Proxy): A popular web application security scanner that helps in finding vulnerabilities in web apps, such as XSS, SQL injection, and others.

Burp Suite: A suite of tools for web application security testing, including features for vulnerability scanning, penetration testing, and manual security assessments.

Metasploit: A penetration testing framework that allows security professionals to develop and execute exploit code against a remote target machine.

Nessus Essentials: A free version of Nessus for conducting vulnerability assessments in a more limited scope, often used in smaller environments.

Snort: Open-source intrusion detection and prevention system that monitors network traffic for signs of malicious activity.

OpenVAS (Greenbone): An open-source vulnerability scanner that identifies vulnerabilities across various services and systems.

Qualys: Cloud-based vulnerability management platform that automates vulnerability scanning and helps organizations stay compliant with industry standards.

Netcat: A network utility used for debugging and scanning network connections, often used in penetration testing for its ability to read and write data across network connections.

John the Ripper: A fast password cracking tool used to assess password strength and help in identifying weak or easily guessed passwords.

Hydra: A network login cracker that supports multiple protocols, used for brute-force password testing and auditing.

Sysinternals Suite: A collection of tools for advanced system monitoring and management, particularly useful for identifying and analyzing security issues on Windows systems.

Nikto: A web server scanner that helps identify various vulnerabilities, including outdated software, security misconfigurations, and common flaws.

Sleuth Kit: A suite of digital forensics tools for investigating disk images, file systems, and data to detect malicious activity and recover evidence.

Cuckoo Sandbox: A malware analysis tool that allows you to run files in an isolated environment to analyze their behavior and identify malicious activity.

AlienVault OSSIM: Open-source SIEM (Security Information and Event Management) platform for monitoring and analyzing security incidents, providing threat detection and response capabilities.
