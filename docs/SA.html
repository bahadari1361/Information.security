Security Assessment for the Central Bank of Afghanistan

As the Information Security Manager at the Central Bank of Afghanistan, I led comprehensive security assessments to evaluate the integrity and security posture of the bank's IT infrastructure. These assessments were essential for identifying potential vulnerabilities and ensuring the protection of sensitive financial data. Using a combination of manual and automated testing tools, I conducted thorough penetration testing, vulnerability scans, and risk analysis to assess the security landscape and provide actionable recommendations for improvement.

  • Identify vulnerabilities in the bank's internal and external networks.
  • Evaluate the security of critical systems and applications.
  • Ensure compliance with industry standards.
  • Develop and implement strategies to address identified risks.
  • Nessus: Vulnerability scanning and risk assessment tool.
  • Nmap: Network discovery and security auditing tool.
  • Kali Linux: Penetration testing platform with numerous tools.
  • Acunetix: Automated web application security scanner.
  • Wireshark: Network protocol analyzer for capturing and analyzing network traffic.
  • OWASP ZAP: Web application security scanner.
  • Burp Suite: Suite of tools for web application security testing.
  • Metasploit: Penetration testing framework.
  • Snort: Open-source intrusion detection and prevention system.
  • OpenVAS: Open-source vulnerability scanner.
  • Qualys: Cloud-based vulnerability management platform.
  • Netcat: Network utility for debugging and scanning network connections.
  • John the Ripper: Password cracking tool.
  • Hydra: Network login cracker.
  • Sysinternals Suite: Tools for advanced system monitoring.
  • Nikto: Web server scanner.
  • Sleuth Kit: Suite of digital forensics tools.
  • Cuckoo Sandbox: Malware analysis tool.
  • AlienVault OSSIM: Open-source SIEM platform.
  • Preparation: Defined the scope and objectives of the security assessment, identifying systems and applications to be tested.
  • Vulnerability Scanning: Performed automated scans using various tools like Nessus and Qualys.
  • Penetration Testing: Utilized tools like Kali Linux and Metasploit for simulated cyber-attacks.
  • Risk Assessment: Prioritized vulnerabilities based on severity and business impact.
  • Remediation Support: Collaborated with IT teams to address vulnerabilities and improve security.
