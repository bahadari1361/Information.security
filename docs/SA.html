    <button class="section-btn" onclick="toggleCollapse('section3')">Tools Used</button>

    <div id="section3" class="content">

        <p><strong>Nessus:</strong> Vulnerability scanning and risk assessment tool to identify known security flaws and weaknesses across systems and networks.</p>

        <p><strong>Nmap:</strong> Network discovery and security auditing tool that helps identify open ports, running services, and network configurations.</p>

        <p><strong>Kali Linux:</strong> A penetration testing platform equipped with numerous tools for vulnerability assessment and exploitation.</p>

        <p><strong>Acunetix:</strong> Automated web application security scanner that detects vulnerabilities like SQL injection, XSS, and other web-based security risks.</p>

        <p><strong>Wireshark:</strong> Network protocol analyzer for capturing and analyzing network traffic to identify potential vulnerabilities in network protocols and communication.</p>

        <p><strong>OWASP ZAP (Zed Attack Proxy):</strong> A popular web application security scanner that helps in finding vulnerabilities in web apps, such as XSS, SQL injection, and others.</p>

        <p><strong>Burp Suite:</strong> A suite of tools for web application security testing, including features for vulnerability scanning, penetration testing, and manual security assessments.</p>

        <p><strong>Metasploit:</strong> A penetration testing framework that allows security professionals to develop and execute exploit code against a remote target machine.</p>

        <p><strong>Nessus Essentials:</strong> A free version of Nessus for conducting vulnerability assessments in a more limited scope, often used in smaller environments.</p>

        <p><strong>Snort:</strong> Open-source intrusion detection and prevention system that monitors network traffic for signs of malicious activity.</p>

        <p><strong>OpenVAS (Greenbone):</strong> An open-source vulnerability scanner that identifies vulnerabilities across various services and systems.</p>

        <p><strong>Qualys:</strong> Cloud-based vulnerability management platform that automates vulnerability scanning and helps organizations stay compliant with industry standards.</p>

        <p><strong>Netcat:</strong> A network utility used for debugging and scanning network connections, often used in penetration testing for its ability to read and write data across network connections.</p>

        <p><strong>John the Ripper:</strong> A fast password cracking tool used to assess password strength and help in identifying weak or easily guessed passwords.</p>

        <p><strong>Hydra:</strong> A network login cracker that supports multiple protocols, used for brute-force password testing and auditing.</p>

        <p><strong>Sysinternals Suite:</strong> A collection of tools for advanced system monitoring and management, particularly useful for identifying and analyzing security issues on Windows systems.</p>

        <p><strong>Nikto:</strong> A web server scanner that helps identify various vulnerabilities, including outdated software, security misconfigurations, and common flaws.</p>

        <p><strong>Sleuth Kit:</strong> A suite of digital forensics tools for investigating disk images, file systems, and data to detect malicious activity and recover evidence.</p>

        <p><strong>Cuckoo Sandbox:</strong> A malware analysis tool that allows you to run files in an isolated environment to analyze their behavior and identify malicious activity.</p>

        <p><strong>AlienVault OSSIM:</strong> Open-source SIEM (Security Information and Event Management) platform for monitoring and analyzing security incidents, providing threat detection and response capabilities.</p>

    </div>

            <li><strong>Risk Assessment:</strong> Prioritized vulnerabilities based on their severity and business impact, providing a clear risk level for each identified issue.</li>

            <li><strong>Remediation Support:</strong> Collaborated with IT teams to address vulnerabilities and implement security improvements, ensuring that critical vulnerabilities were mitigated.</li>
